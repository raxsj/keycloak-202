# Lab 01 — Crear API Node

## Paso 01 — Inicializar el proyecto Node

---

## 🎯 Objetivo

Crear la base del backend Node.js que utilizaremos para validar tokens emitidos por Keycloak.

---

## 📍 Ubicación

Desde la raíz del repositorio:

```
apps/api-node
```

---

## 📦 Inicializar proyecto

Ejecutar:

```bash
npm init -y
```

---

## 📁 Verificar estructura

Debe generarse:

* package.json

---

## 🧱 Crear estructura básica

Crear archivo principal:

```
index.js
```

Contenido mínimo temporal:

```javascript
console.log('API Node initialized');
```

---

## ▶️ Probar ejecución

Ejecutar:

```bash
node index.js
```

Debe mostrarse:

```
API Node initialized
```
